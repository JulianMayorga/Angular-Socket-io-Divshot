# Connect Four Frontend

## Getting started

### Prerequisites

* Node.js
* NPM
* Divshot CLI: `npm install -g divshot-cli`
* Bower: `npm install -g bower`

### Installation

`npm install && bower install`

## Development

`gulp serve`

## Testing

* Unit tests: `gulp test`
* Integration tests: `gulp protractor`

## Deployment

`gulp build && divshot push`
